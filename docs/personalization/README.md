
## 🤔 常见问题

**Q: 必须按顺序实施吗？**
A: 不必。第一层必须，其他可以灵活组合。

**Q: E盘机械硬盘够用吗？**
A: 完全够用。即使10万条对话，向量数据只有~10MB。

**Q: 必须微调模型吗？**
A: 不必。前三层已经能获得很好的效果，微调只是锦上添花。

**Q: 5090能微调多大的模型？**
A: 7B-14B模型都可以，推荐Qwen2.5-7B或GLM-4-9B。

**Q: 需要多少数据？**
A:
- 第一层: 0（立即可用）
- 第二层: 自动积累
- 第三层: 建议1000条高质量对话
- 第四层: 自动学习

**Q: 效果能到什么程度？**
A:
- 第一层: 相当于Character.AI水平
- 第二层: 相当于有长期记忆的Character.AI
- 第三层: 专业级定制
- 第四层: 主动适应你的AI伙伴
